Backend Developer
-----------------

I am a backend developer specializing in building RESTful APIs using Python and Django. My expertise includes database management, authentication, API development, and integration of geospatial technologies into backend systems. I have hands-on experience working with Django REST Framework (DRF), PostgreSQL, and cloud deployment. I am passionate about designing scalable, secure, and efficient backend systems that power web applications. I also have a strong foundation in geospatial data processing, enabling me to build APIs that integrate with GIS databases and mapping services. You can check out my projects on GitHub: https://github.com/olaz20.
