# 👋 Hi, I'm Olamoyegun Olamide Samuel  

## 🚀 Backend Developer | FastAPI • Django • JavaScript

I specialize in building **scalable, secure, and efficient RESTful APIs** using **Python (FastAPI & Django)**. My work includes:

- 🔐 **Authentication & Authorization** (JWT, OAuth2)
- 🧩 **Microservices & Modular API Design**
- 🌍 **Geospatial Services** (GIS APIs, Map Routing)
- ⚙️ **Payment Integration** (Paystack, Stripe)
- ☁️ **Cloud & DevOps** (Render, Heroku, AWS EC2/S3)
- 📦 **Database Management** (PostgreSQL, SQLite)
- 🔁 **Asynchronous Processing** with `async`/`await`

I also use **JavaScript** to integrate APIs into real-time dashboards, and automation tools.

> ⚡ I aim to build intelligent backend systems for SaaS, logistics, and real estate platforms.
